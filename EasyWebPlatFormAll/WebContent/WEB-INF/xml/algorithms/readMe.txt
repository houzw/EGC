frequencySampler.xml是将HM Env.Layers ManageMent 和FS Env.Layers ManageMent合并后的文档。
frequencySampler_divide.xml是将HM Env.Layers ManageMent 和FS Env.Layers ManageMent分开后的文档。
上述两个文件最大的区别在于
        frequencySampler.xml中的内容包含：
		<DataName>HM Env.Layers ManageMent</DataName>
		<DataType>Manager</DataType>
		<DataSematic>HM Env.Layers ManageMent</DataSematic>
		
		frequencySampler_divide.xml中的内容包含：
		<DataName>FS Env.Layers ManageMent</DataName>
		<DataType>Manager</DataType>
		<DataSematic>FS Env.Layers ManageMent</DataSematic>
